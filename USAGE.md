# 宴席菜品配料统计系统 - 用户使用说明书

## 软件简介

宴席菜品配料统计系统是一款专为餐饮行业设计的食材成本计算软件。通过建立食材库、菜品库和宴席菜单，帮助您快速计算宴席所需的食材用量和成本，提高采购效率，控制成本支出。

## 系统要求

- Windows 7 或更高版本
- 至少 100MB 可用磁盘空间
- 建议分辨率：1024×768 或更高

## 安装与启动

1. **解压软件包**：将下载的软件包解压到您希望安装的文件夹
2. **启动程序**：双击 `DishWeight.exe` 文件启动程序
3. **首次运行**：程序会自动创建数据文件，无需额外配置

> **注意**：请确保程序文件夹有读写权限，避免安装在系统保护目录下

## 主界面介绍

程序启动后，您会看到一个包含四个选项卡的主界面：

- **食材管理**：管理所有食材的基本信息
- **菜品管理**：创建菜品配方和配料清单
- **宴席管理**：组织宴席菜单和份数
- **统计分析**：计算食材用量和成本

## 使用流程

### 第一步：建立食材库

1. 点击"食材管理"选项卡
2. 在输入框中填写：
   - **食材名称**：如"猪肉"、"大米"、"青椒"等
   - **单位**：如"斤"、"公斤"、"个"、"袋"等
   - **单价**：每单位的价格（元）
3. 点击"添加食材"按钮保存

**操作技巧**：
- 可以选择表格中的食材进行修改或删除
- 建议按食材类别分批添加，如蔬菜类、肉类、调料类等
- 单价可以根据市场变化随时更新

### 第二步：创建菜品配方

1. 点击"菜品管理"选项卡
2. 点击"新建菜品"按钮
3. 填写菜品名称，如"红烧肉"、"宫保鸡丁"等
4. 添加配料：
   - 从下拉菜单选择食材
   - 输入该食材的用量
   - 点击"添加配料"
5. 重复步骤4，添加所有配料
6. 点击"保存菜品"完成创建

**配方示例**：
- 菜品：红烧肉
- 配料：猪肉 2斤，生抽 0.1斤，老抽 0.05斤，冰糖 0.1斤

### 第三步：组织宴席菜单

1. 点击"宴席管理"选项卡
2. 点击"新建宴席"按钮
3. 填写宴席名称，如"婚宴"、"生日宴"、"公司年会"等
4. 添加菜品：
   - 从下拉菜单选择菜品
   - 输入该菜品的份数
   - 点击"添加菜品"
5. 重复步骤4，添加所有菜品
6. 点击"保存宴席"完成创建

**菜单示例**：
- 宴席：婚宴（10桌）
- 菜品：红烧肉 10份，宫保鸡丁 10份，清蒸鱼 10份

### 第四步：计算食材用量

1. 点击"统计分析"选项卡
2. 从下拉菜单选择要计算的宴席
3. 点击"计算食材用量"按钮
4. 查看统计结果：
   - 每种食材的总用量
   - 每种食材的单价
   - 每种食材的总价
   - 整个宴席的总成本

## 数据管理

### 数据保存
- 程序会自动保存所有操作，无需手动保存
- 数据保存在程序目录下的 `dish_data.json` 文件中

### 数据备份
- 建议定期复制 `dish_data.json` 文件到其他位置作为备份
- 可以通过"导出Excel"功能导出数据报表

### 数据查看
- 点击"查看数据信息"按钮可查看当前数据统计
- 显示食材、菜品、宴席的数量和文件信息

## 导出功能

程序支持将数据导出为Excel文件，方便打印和进一步编辑：

1. 在"统计分析"选项卡中点击"导出Excel"
2. 选择保存位置和文件名
3. 导出的Excel文件包含三个工作表：
   - **食材库**：所有食材的详细信息
   - **菜品配方**：每道菜的配料清单
   - **宴席菜单**：宴席中的菜品和份数

## 实用技巧

### 食材管理技巧
- **分类管理**：可以在食材名称前加前缀，如"蔬菜-白菜"、"肉类-猪肉"
- **单位统一**：建议同类食材使用相同单位，便于计算
- **价格更新**：定期根据市场价格更新食材单价

### 菜品配方技巧
- **标准化配方**：制定标准的菜品配方，确保口味一致
- **成本控制**：通过调整配料比例控制菜品成本
- **季节调整**：根据季节调整配方中的蔬菜种类

### 宴席规划技巧
- **提前规划**：提前制定宴席菜单，便于采购安排
- **成本预算**：通过系统计算控制宴席总成本
- **余量考虑**：实际采购时可在计算结果基础上增加5-10%余量

## 常见问题

### Q：如何修改已有的食材信息？
A：在食材管理页面，点击表格中的食材行，信息会自动填入输入框，修改后点击"更新食材"。

### Q：如何删除不需要的菜品？
A：在菜品管理页面，选择要删除的菜品，点击"删除菜品"按钮。注意：如果菜品已被宴席使用，需要先从宴席中移除。

### Q：计算结果不准确怎么办？
A：请检查食材单价是否正确，菜品配方是否完整，宴席菜品份数是否正确。

### Q：如何备份我的数据？
A：复制程序目录下的 `dish_data.json` 文件到安全位置，或使用"导出Excel"功能。

### Q：程序无法启动怎么办？
A：确保程序文件夹有读写权限，尝试以管理员身份运行程序。

## 注意事项

1. **数据安全**：定期备份数据文件，避免意外丢失
2. **权限要求**：确保程序目录有读写权限
3. **操作顺序**：建议按照"食材→菜品→宴席→统计"的顺序操作
4. **删除提醒**：删除食材前确保没有菜品在使用，删除菜品前确保没有宴席包含该菜品

## 技术支持

如果您在使用过程中遇到问题，请：
1. 首先查阅本说明书的常见问题部分
2. 检查数据文件是否完整
3. 尝试重启程序
